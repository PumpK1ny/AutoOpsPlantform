# 主工作流程

## 1. 调用新闻采集智能体

发送以下内容：

```markdown
# 新闻采集智能体工作流程

1. 使用 `get_fund_news` 和 `get_fast_news` 获取基金新闻和快速新闻的概览，并根据重要程度筛选 5-8 篇新闻，利用 `get_multi_page_news` 一次性获取完具体的新闻内容。

2. 先使用 `get_dongfang_notice` 获取东方基金的公告，选取重要的 3-5 条公告，再使用 `read_dongfang_notice` 获取具体的公告内容。

3. 使用 `get_hot_article_ranking` 获取近期热点排名的文章。

4. 根据先前几步获取的信息，总结出近期基金的动态和趋势，用专业易懂的语言撰写总结报告，保存为 Markdown 格式。

5. 最后只回答下面的内容，不得进行总结和输出其他内容：
   - 总结报告已保存为 `data/result/news/yyyy-mm-dd.md`。
```

## 2. 调用基金分析智能体

发送以下内容：

```markdown
# 基金分析智能体工作流程

1. 使用 `get_sector_fund_flow` 获取今日资金流向数据，以及 5 日资金流向数据。

2. 根据资金流向数据，分析基金的动态和趋势。

3. 最后只回答下面的内容，不得进行总结和输出其他内容：
   - 分析报告已保存为 `data/result/fund_analysis/yyyy-mm-dd.md`。

## 要求

- 不要深度思考每个板块的资金流向数据，只需要根据资金流向数据，分析基金的动态和趋势。
- 禁止过度思考。
```

## 3. 综合分析与报告生成

- **深度解读**：结合新闻采集智能体的消息层面分析（政策动向、市场情绪、公告事件）与基金分析智能体的技术层面分析（资金流向、板块轮动、机构行为），进行交叉验证与关联分析。

- **矛盾识别**：识别消息层面与技术层面的矛盾点（如利好消息但资金流出、情绪悲观但机构抄底），分析可能原因并给出判断依据。

- **趋势研判**：基于双维度数据，研判短期（1-3 天）、中期（1-2 周）市场趋势，明确方向判断与置信度。

- **投资机会**：提炼具体可执行的投资建议，包括关注板块、回避板块、仓位建议、风险提示。

- **报告撰写**：生成结构化的《基金市场综合分析报告》，包含执行摘要、消息解读、资金分析、趋势研判、投资建议、风险提示六大模块。

- **保存路径**：`data/result/final/yyyy-mm-dd.md`
